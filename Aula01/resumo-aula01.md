1. Definição de Algoritmo
O algoritmo é uma sequência bem definida de passos, que devem ser finitos, e tem como objetivo a resolução de algum problema.

2. O que é entrada, processamento e saída
A entrada é onde os dados que serão processados são inseridos; o processamento é onde os dados são manipulados para que se tenha o resultado esperado; a saída é onde o resultado do processamento é apresentado.

3. Um exemplo do dia a dia em forma de passos
Exemplo: Colocar a comida dos gatos, com as vasilhas já vazias e limpas.
1. Recolho as vasilhas
2. Separo a quantidade de ração de cada gato em cada vasilha
3. Coloco as vasilhas nos lugares em que eles comem
4. Busco os gatos
5. Coloco cada gato em sua vasilha
6. Aguardo os gatos comerem
7. Se terminarem, recolho as vasilhas e as lavo

1. Algoritmo - é o passo a passo finito desenvolvido para a resolução de algum problema.
2. GitHub - é uma plataforma, espécie de rede social, em que o que se disponibiliza online seus projetos do Git.

5. Lógica - é o estudo do "pensar corretamente", que nos ajuda a transformar esse pensamento em uma sequência lógica de ações para se obter o resultado esperado.
